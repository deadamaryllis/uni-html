Cursul 12 - CSS

Stilizarea imaginilor cu CSS

Proprietate Descriere Exemplu CSS
border-radius Rotunjire generala a tuturor colturilor border-radius: 15px;
border-left border-right border-top border-bottom Adauga margine in jurul elementului in partea specificata border-right: 6px solid yellow;
border-top-left-radius border-top-right-radius border-bottom-right-radius border-bottom-left-radius Rotunjire colturi specificate border-top-left-radius: 10px;
border Adauga margine in jurul elementului border: 2px solid black;
opacity Seteaza transparenta (0 - invizibil, 1 - opac complet) opacity: 0.5;

Pozitionarea elementelor

Valoare Descriere Exemplu CSS
static Pozitionare implicita position: static
relative Pozitionare relativa position: relative; top: 10px;
absolute Pozitionare absoluta position: absolute; left: 50px;
fixed Pozitionare fixa position: fixed; top: 0;

Pseudo-clase CSS

Pseudo-clasa Descriere Exemplu CSS
:first-child Stil aplicat primului copil p:first-child { color: blue; }
:last-child Stil aplicat ultimului copil li:last-child { font-weight: bold; }
:hover Stil la mouse peste element a: hover { text- decoration: underline; }
:focus Stil la focus pe input input:focus { background-color: yellow }
:link Link nevizitat a:link { color: green; }
:visited Link vizitat a:visited { color: purple; }

Pseudo-elemente CSS

Pseudo-element Descriere Exemplu CSS
::before Adauga continut inainte p::before { content: ">> "; }
::after Adauga continut dupa p::after { content: " <<"; }
::first-letter Stilizeaza prima litera p::first-letter { font-size: 200%; }
::first-line Stilizeaza prima linie p::first-line { font-weight: bold; }

Animatii CSS

Proprietate Descriere Exemplu CSS
animation-name Numele animatiei animation-name: slide;
animation-duration Durata animatiei animation-duration: 4s;
animation-delay Intarziere start animation-delay: 2s;
animation-iteration-count Numar de repetari animation-iteration-count: infinite;
animation-direction Sensul animatiei animation-direction: alternate;
animation-fill-mode Stiluri dupa final animation-fill-mode: forwards;
animation-play-state Pauza/rulare animation-play-state: running;
animation-timing-function Curba de viteza animation-timing-function: ease-in-out;

Tranzitii CSS

Proprietate Descriere Exemplu CSS
transition-property Ce proprietate se anima transition-property: background-color;
transition-duration Durata efectului transition-duration: 1s;
transition-timing-function Tipul de viteza transition-timing-function: ease-in;
transition-delay Intarziere pornire transition-delay: 0.5s;
