Cursul 8 - Formulare

Tag-urile principale

  • <form method=post sau get>. Post adauga datele in link, Get face un http request.
  • <input> - text, password (text cu caractere ascunse), radio buttons (de selectare a unei singuri optiuni), checkbox (de selectare a multiple optiuni)
  • <label> - Textul ce acompaniaza optiunile / butoanele. atributul for asociaza label-ul cu id-ul inputului.
  • <select> - putem selecta din lista dropdown
  • <option> - optiunile din lista
  • <textarea> - un camp de text mai mare, cu mai multe linii
  • butoane: submit - trimite, reset - sterge, button - buton simplu

Cursul 9

  • input type="file" trimite fisiere catre server
  • input accept="image/png, image/jpeg" accepta doar tipuri specifice de fisiere (in acest caz, imagini html sau jpeg)
  • input multiple accepta mai multe fisiere

Tipurile disponibile pentru elementul input:

  • date - selectare dată
  • datetime-local - selectare dată și oră fără fus orar
  • email - validare automată pentru adrese de email
  • month - selectare lună și an
  • number - introducere de numere
  • range - slider pentru valori numerice
  • search - câmp special pentru căutare
  • tel - introducere numere de telefon
  • time - selectare oră
  • url - validare automată pentru adrese web
  • week - selectare săptămână dintr-un an

Pe lângă acestea, există și alte tipuri de input precum checkbox, color, file, hidden, image, password, radio, reset, submit, fiecare având un rol bine definit în funcție de contextul formularului.
