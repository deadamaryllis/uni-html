Detalii curs

HTML (HyperText Markup Language) este limbajul de bază folosit pentru a structura conținutul unei pagini web. Stilizarea acestui conținut este realizată prin CSS (Cascading Style Sheets), care permite aplicarea de culori, fonturi și aranjamente vizuale. De exemplu, cu ajutorul proprietății text-align putem alinia textul la stânga, dreapta sau centra. În acest text, avem aliniere justify pentru o prezentare uniformă. Spațiul dintre litere (letter-spacing) și dintre cuvinte (word-spacing) a fost mărit pentru o mai bună lizibilitate. Rândurile sunt aerisite prin line-height, iar prima linie a fost indentată cu text-indent. Simbolurile precum H₂O (pentru apa) sau formule de tipul a² + b² = c² demonstrează folosirea atributiei vertical-align pentru subscript și superscript.
