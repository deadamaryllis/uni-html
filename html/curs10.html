Cursul 10 - CSS

Modalitati de aplicare CSS

  • Inline - stilizare direct in tag
  • Embedded - stilizare in head, cu tagul style
  • External - stilizare in fisier separat, cu tagul link

Sintaxa CSS

  • selector { proprietate: valoare; }

Exemplu:

  • h1 { font-size: 24px; color: blue; }

Selectori CSS

  • Tag - selecteaza toate tagurile de acelasi tip
  • Class - selecteaza toate tagurile cu aceeasi clasa (punct)
  • Id - selecteaza un singur tag (hashtag)
  • Descendent - selecteaza un tag dintr-un alt tag (spatiu)

Exemplu de selectori CSS

  • Exemplu: div p - selecteaza toate tagurile p dintr-un div
  • #id - selecteaza un element cu id-ul dat
  • .clasa - selecteaza toate elementele cu clasa data
  • div.clasa - selecteaza toate div-urile cu clasa data
  • div#id - selecteaza div-ul cu id-ul dat

Prioritati CSS

  • Inline - cele mai importante
  • Embedded - prioritate medie
  • Externe - prioritate mai mica
  • Daca doua reguli au aceeasi specificitate, cea scrisa mai tarziu in cod are prioriate.
  • Selectori mai specifici (ex. ID) au prioritate mai mare decat clasele sau etichetele.
  • Selectori mai lungi si mai specifici sunt preferati in fata celor generali.
  • !important - prioritate maxima, dar nu este recomandat sa fie folosit.

Etichete HTML utile in CSS

  • span - folosit pentru a stiliza portiuni de text inline, fara intrerupere de linie
  • div - defineste o zona de tip block, separata vizual in pagina.

Fundaluri (background)

  • background-image - imagine de fundal
  • background-color - culoare de fundal
  • background-repeat - repeta imaginea de fundal
  • background-position - pozitia imaginii de fundal
  • background-size - dimensiunea imaginii de fundal
  • background-attachment - cum se comporta imaginea de fundal la scroll

Formatarea textului

  • color - culoarea textului
  • font-family - fontul textului
  • font-size - dimensiunea fontului
  • font-weight - grosimea fontului
  • text-align - alinierea textului (left, right, center)
  • text-decoration - decorarea textului (underline, overline)
  • text-transform - transformarea textului (uppercase, lowercase)
  • line-height - inaltimea liniei
  • letter-spacing - spatiul dintre litere
  • word-spacing - spatiul dintre cuvinte
  • font-style - stilul fontului (italic, oblique)

Alte proprietati utile

  • margin - marginea elementului
  • padding - spatiul interior al elementului
  • border - conturul elementului
  • width - latimea elementului
  • height - inaltimea elementului
  • display - modul de afisare al elementului (block, inline, flex)
  • position - pozitionarea elementului (static, relative, absolute)
  • z-index - ordinea de suprapunere a elementelor
  • overflow - cum se comporta continutul care depaseste dimensiunile elementului
  • opacity - transparenta elementului
  • box-shadow - umbra elementului
  • text-shadow - umbra textului
  • font - permite o setare rapida a mai multor proprietati (style, weight, transform, size, family)
