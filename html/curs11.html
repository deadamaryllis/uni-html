Cursul 11 - CSS

Configurarea textului

Proprietate Descriere Valori admise
letter-spacing Spatiu intre litere normal, 0, valori pozitive/negative (px, em)
word-spacing Spatiu intre cuvinte normal, valori pozitive/negative (px, em)
line-height Spatiu intre linii valori numerice, procentuale, normal
text-align Aliniere orizontala left, right, center, justify
vertical-align Aliniere verticala super, sub, baseline, top, middle, bottom, %
text-indent Indentare prima linie valori pozitive / negative (px, em, %)

Box Model CSS

Proprietate Descriere Valori admise
margin (top/bottom/left/right) Spatiu exterior element auto, valori numerice, procentuale
padding (top/bottom/left/right) spatiu interior fata de border valori numerice, procentuale
border (top/bottom/left/right) bordura completa grosime (0, thin, medium thick, px), stil, culoare
width, height dimensiuni element valori numerice, procentuale, auto

Stilizarea linkurilor

Selector Descriere Valori admise
a:link stil link nevizitat orice proprietate CSS aplicabila (ex. color)
a:visited stil link vizitat idem
a:hover stil la trecerea mouse-ului idem

Liste

Proprietate Descriere Valori admise
list-style-type Tip marcator disc, circle, square, none, decimal, roman, alpha
list-style-image Imagine ca marcator url(imagine)
list-style-position Pozitia marcatorului inside, outside
list-style Combina toate cele de mai sus tip, pozitie, imagine

Tabele

Proprietate Descriere Valori admise
border-collapse contopirea contururilor collapse, separate
border-spacing distanta dintre celule valori numerice (px, em)
caption-side pozitia titlului tabelului top, bottom
empty-cells afisarea celulelor goale show, hide
table-layout sablonul tabelului auto, fixed
