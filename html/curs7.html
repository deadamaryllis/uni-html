Cursul 7 - Tabele

Tag-urile principale

  • <table>
  • <tr> - table row
  • <td> - table data
  • colspan - întindem celula pe mai multe coloane
  • rowspan - întindem celula pe mai multe rânduri
  • cellspacing și cellpadding - distanța și spațiul dintre celule / marginile celulelor
  • putem modifica mărimea tabelului cu width și height
  • putem alinia tabelul cu align
  • putem pune tabele înauntru altor tabele
  • putem modifica stilul border-ului
  • putem modifica culorile celulelor
  • putem pune imagini înauntru tabelelor
